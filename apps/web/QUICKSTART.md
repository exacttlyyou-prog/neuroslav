# Быстрый старт

## Установка и запуск

```bash
# Перейти в директорию проекта
cd apps/web

# Установить зависимости (если еще не установлены)
npm install

# Запустить dev server
npm run dev
```

Приложение будет доступно по адресу: **http://localhost:3000**

## Что должно работать

1. Откроется главная страница с редиректом на `/tasks`
2. Три таба в навигации:
   - **Task Reminder** - форма для создания задач
   - **Meeting Dump** - форма для загрузки встреч
   - **Context/RAG** - форма для загрузки документов

## Возможные проблемы

### Порт 3000 занят
Измените порт в `package.json`:
```json
"dev": "next dev -p 3001"
```

### Ошибки компиляции
Убедитесь, что все зависимости установлены:
```bash
rm -rf node_modules package-lock.json
npm install
```

### Стили не применяются
Проверьте, что `app/globals.css` импортирован в `app/layout.tsx`
