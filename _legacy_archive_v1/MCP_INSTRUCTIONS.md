# MCP Notion - Инструкция по получению контента встречи

## Автоматический способ:
1. Используйте команду в терминале Cursor:
   ```
   curl -X POST http://localhost:3000/mcp/notion/fetch \
     -H "Content-Type: application/json" \
     -d '{"id": "2edfa7fd637180b98715fa9f348f90f9"}'
   ```

2. Или используйте MCP инструмент напрямую в коде (если настроен MCP сервер)

## Ручной способ через Cursor:
1. В терминале Cursor выполните команду для получения данных страницы
2. Скопируйте JSON ответ
3. Сохраните его в файл `mcp_response.json` в корне проекта
4. Запустите скрипт повторно - он автоматически найдет и обработает файл

## Структура MCP ответа:
- Содержит блоки `<meeting-notes>` с транскрипциями
- Последний блок = самая свежая встреча
- Скрипт автоматически извлекает контент из последнего блока
